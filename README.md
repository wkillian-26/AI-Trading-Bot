# AI Trading Bot

An advanced machine-learning trading research environment with:

- Feature engineering (SMA, volatility, RSI)
- Gradient boosting ML model (HistGradientBoostingClassifier)
- Out-of-sample backtesting
- Research notebook for rapid experimentation
